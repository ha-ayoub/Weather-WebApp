**FR** · [EN](#-skywatcher--english)

</div>

---

## 🇫🇷 Présentation

**Skywatcher** est une application météo vanilla JS pensée pour être à la fois fonctionnelle et visuellement soignée. Elle affiche en temps réel les conditions météorologiques d'une ville, les prévisions sur 7 jours et une courbe de température sur 72 heures.

---

## ✨ Fonctionnalités

- 🔍 **Recherche de ville** — Saisie libre avec validation à la touche `Entrée` ou via le bouton
- 📍 **Géolocalisation automatique** — Détection de la position de l'utilisateur via l'API du navigateur
- 🌡️ **Météo actuelle** — Température, ressenti, humidité, vent, visibilité, min/max
- 📅 **Prévisions 7 jours** — Cartes journalières avec icône météo, température max et min
- 📈 **Graphique de température** — Courbe interactive sur 72h affichant la température réelle et le ressenti (Chart.js)
- 🌙 **Mode sombre / clair** — Thème togglable avec persistance via `localStorage` et respect du thème système
- 🔔 **Notifications toast** — Retours visuels animés pour les erreurs et confirmations
- 📱 **Responsive** — Interface adaptée mobile, tablette et desktop

## 🇬🇧 Overview

**Skywatcher** is a vanilla JS weather app built to be both functional and visually refined. It displays real-time weather conditions for any city, a 7-day forecast, and a 72-hour temperature chart.

---

## ✨ Features

- 🔍 **City search** — Free text input with `Enter` key or button validation
- 📍 **Auto geolocation** — Detect the user's current position via the browser Geolocation API
- 🌡️ **Current weather** — Temperature, feels like, humidity, wind speed, visibility, min/max
- 📅 **7-day forecast** — Daily cards with weather icon, max and min temperature
- 📈 **Temperature chart** — Interactive 72h line chart showing temperature and feels like (Chart.js)
- 🌙 **Dark / light mode** — Toggleable theme with `localStorage` persistence and system theme detection
- 🔔 **Toast notifications** — Animated visual feedback for errors and confirmations
- 📱 **Responsive** — Fully adapted for mobile, tablet and desktop
